CICATRIZES FICHA - ARQUIVOS INICIAIS DA EXTENSÃO OWLBEAR RODEO

O que existe nesta pasta:
- manifest.json -> arquivo principal que o Owlbear Rodeo lê
- ficha.html -> tela inicial da ficha
- icon.svg -> ícone da extensão

Como usar:
1. Crie um repositório no GitHub.
2. Envie estes 3 arquivos para a raiz do repositório.
3. Ative o GitHub Pages nas configurações do repositório.
4. Pegue a URL pública do manifest.json.
5. No Owlbear Rodeo, adicione a extensão usando essa URL.

Exemplo de URL final:
https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/manifest.json
